# Fleet Plugins Marketplace

Official plugin repository for the Fleet ecosystem (Claude Code).

## Available Plugins

| Plugin | Description | Version |
|--------|-------------|---------|
| [fleet-dev](plugins/fleet-dev/) | Development Operations - Setup, Linting, Obsidian, DSGVO, Deployment | 1.0.0 |

## Installation

```bash
# Add marketplace
claude plugin add-marketplace https://github.com/dnhrdt/fleet-plugins

# Install plugin
claude plugin install fleet-dev
```
