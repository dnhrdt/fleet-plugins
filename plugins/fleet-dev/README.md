# fleet-dev Plugin

Development Operations Plugin for Fleet projects.

## Skills

| Skill | Description |
|-------|-------------|
| setup | Project initialization with Memory Bank, Git, Tools |
| linting | Code quality enforcement (Python, JavaScript, Shell) |
| obsidian | Git-to-Obsidian sync via post-commit hook |
| dsgvo | Customer data protection check before public commits |
| deployment | Release packaging and deployment workflows |

## Tools

| Tool | Description |
|------|-------------|
| obsidian-sync | Post-commit hook for Obsidian Vault sync |
| linting-system | Pre-configured linting configs |
| dsgvo-check | Customer data detection scripts |

## Installation

```bash
claude plugin install fleet-dev
```

## Usage

After installation, invoke skills with:
- "Load setup skill"
- "Load linting skill"
- etc.
